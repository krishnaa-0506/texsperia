🆘 TROUBLESHOOTING:

Q: Port already in use?
A: Kill the process:
   netstat -ano | findstr :3001  (Windows)
   lsof -ti:3001 | xargs kill -9 (Mac/Linux)

Q: MongoDB connection error?
A: Check api/.env MONGODB_URI value

Q: How to change admin password?
A: Edit api/.env → ADMIN_PASSWORD

Q: Both frontend and backend not starting?
A: Check npm install completed successfully
