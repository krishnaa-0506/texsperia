🚀 NEXT STEPS (IMPORTANT):

1. ADD YOUR MONGODB URL
   📁 Open: server\.env
   ✏️ Replace: MONGODB_URI with your actual MongoDB connection string
   
   Get MongoDB URL from:
   🌐 https://www.mongodb.com/cloud/atlas
   
   Example format:
   mongodb+srv://username:password@cluster.mongodb.net/texperia

2. RUN SETUP (ONE TIME)
   Open PowerShell and run:
   .\setup.ps1

3. START THE APPLICATION
   Run:
   .\start.ps1
   
   Or manually:
   Terminal 1: cd server; npm start
   Terminal 2: npm run dev

4. ACCESS THE WEBSITE
   🌐 http://localhost:5173
   
5. ACCESS ADMIN PANEL
   Click yellow shield icon (bottom-right)
   Password: hkkk@admin2026
